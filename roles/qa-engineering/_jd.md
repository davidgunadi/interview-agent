# QA Engineer

## Specific Skill & Qualification

- Test Case Design & Analysis
- API Testing and Debugging
- Release Validation

## Job Description

Bertanggung jawab memastikan kualitas produk melalui perencanaan, pelaksanaan, dan peningkatan proses pengujian, serta berkontribusi dalam identifikasi risiko kualitas sejak tahap requirement hingga release.

### 1. Test Planning & Test Execution

- Menganalisis requirement, user story, dan technical design.
- Membuat test scenario dan test case yang mencakup functional, integration, dan regression testing.
- Melakukan manual testing pada aplikasi web, mobile, API, dan backend service.
- Memastikan seluruh acceptance criteria telah tervalidasi sebelum release.

### 2. Defect Management

- Mengidentifikasi, mendokumentasikan, dan melaporkan defect secara jelas dan terstruktur.
- Melakukan root cause analysis awal terhadap defect yang ditemukan.
- Berkolaborasi dengan Developer, Product Manager, dan Technical Lead untuk proses perbaikan issue.
- Melakukan retesting dan regression testing setelah perbaikan dilakukan.

### 3. Release Validation

- Melakukan smoke test dan sanity test sebelum deployment.
- Memastikan fitur yang akan dirilis memenuhi standar kualitas yang ditetapkan.
- Mendukung proses monitoring dan validasi pasca release.
